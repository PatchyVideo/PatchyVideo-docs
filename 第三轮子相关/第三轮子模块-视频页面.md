# 视频页面

*本页面是原帕琪站主页的简化复刻版，为了方便用户对视频标签进行整理，以及检查和维护视频数据*

## 显示内容

### 视频大类

- ALL

- 综合

- MMD/手书
- 音乐/MV
- 科普杂谈
- 游戏/实况
- 动画
- 其他
- 非东方作品

### 视频列表

- 缩略图
- 视频名称
- 分p名（B站）
- 原视频上传时间
- 帕琪站索引时间
- 其他标签
- 列表模式、网格模式调整

### 高级筛选选项

- 来源网站筛选
- 视频列表排序
- 上传时间段查询
- 无标签、标签不全视频显示
- 失效视频显示
- 自动发布视频显示
- 是否使用默认屏蔽标签
- 视频等级筛选（管理员）

### 其他

- 分页
  - 每页显示视频数量
  - 跳转到第几页
- 视频批量选择
  - 批量添加到播放列表
  - 批量编辑共有标签
  - 批量修改等级（管理员）
